# `k`

By NriotHrreion

## 使用

#### Node.js

1. 下载安装k.js

```cmd
npm i @nocp/k
```

2. 引入

```js
const k = require("@nocp/k");
k.is({}, {}); // true
```

#### 浏览器

1. 在`<head>`标签中加入

```html
<script src="https://cdn.jsdelivr.net/npm/@nocp/k/k.js"></script>
```

2. 直接使用

```js
is({}, {}); // true
```
